            |
            | ルール:
            | - summary: 要約・サマリー・概要の依頼（例: 「要約して」「サマリーを出して」「まとめて」「概要を教えて」「3行で」など）
            | - coaching: 商談の進め方、提案方針、競合対策、失注リスク、次回アクション、経営層向け訴求、ヒアリング観点などの相談
            | - unknown: 上記いずれにも明確に該当しない場合
            |
            | 優先順位:
            | - 「競合対策をまとめて」「失注リスクを3行で」のように summary 系と coaching 系が混在する場合は coaching を優先。
            | - 「3行で」は出力形式の指定であり、内容がコーチングであれば coaching に分類。
            |
            | キーワード抽出:
            | - 「要約」「サマリー」「まとめ」「概要」などが含まれる場合は summary_message を非空に設定。
            | - 「コーチング」「壁打ち」「相談」「対策」「次」「アクション」「行動」「整理」などが含まれる場合は next_action_message を非空に設定。

            # ガードロジック（フラット化）
            if @variables.next_action_message is not None and @variables.next_action_message != "":
                set @variables.intent_category = "coaching"

            if (@variables.next_action_message is None or @variables.next_action_message == "") and @variables.summary_message is not None and @variables.summary_message != "":
                set @variables.intent_category = "summary"


            # intent_category を確定
            run @actions.set_intent_category


            # 遷移（フラット化）
            if @variables.intent_category == "unknown":
                | コーチングの相談ですか？それとも会話の要約が必要ですか？
                transition to @subagent.Opportunity_Status_Summary


            if @variables.intent_category == "summary":
                transition to @subagent.Opportunity_Status_Summary


            if @variables.intent_category == "coaching":
                transition to @subagent.Opportunity_Coaching


        actions:
            set_intent_category: @utils.setVariables
                description: "intent_category 変数を分類結果で更新する（必ず最初に呼び出す）。"
                with intent_category=...
            go_to_Opportunity_Coaching: @utils.transition to @subagent.Opportunity_Coaching
                description: "コーチング用サブエージェントへ遷移"
                available when @variables.intent_category == "coaching"
            go_to_Opportunity_Status_Summary: @utils.transition to @subagent.Opportunity_Status_Summary
                description: "サマリー用サブエージェントへ遷移"
                available when @variables.intent_category == "summary"
